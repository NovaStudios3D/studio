# Cybernox3D Editor Features

## Core 3D Engine
- **3D Scene Rendering**: Utilizes Three.js for a high-performance WebGL viewport.
- **Camera Controls**: Full orbit controls for panning, zooming, and rotating the camera.
- **Dynamic Lighting**: A time-of-day system that realistically changes lighting and shadows.
- **Ground & Grid**: A detailed grid on a ground plane for precise object placement.

## Object Management
- **Add Primitive Shapes**: Cube, Sphere, Plane, Pyramid, Cylinder, Monkey, Capsule, Torus, TorusKnot.
- **Add 3D Text**: Create 3D text objects from user input.
- **Select Objects**: Click objects in the scene or in the Object List to select them.
- **Copy/Duplicate**: Clone the selected object with a single action (Ctrl+D).
- **Delete**: Remove selected objects from the scene (Delete key).
- **Hide/Show Objects**: Toggle the visibility of any object using the eye icon in the Object List.
- **Undo/Redo**: Step backward and forward through your actions using dedicated Undo/Redo buttons.

## Transformation & Properties
- **Transform Gizmos**: Manipulate objects with visual gizmos for Move, Rotate, and Scale.
- **Keyboard Shortcuts**: Switch between tools (M, R, S), copy (Ctrl+D), and delete.
- **Properties Panel**: View key properties of the selected object.
- **Property Editor**: A popup dialog to precisely edit an object's name, position, rotation, scale, and color.

## Import & Export
- **Import Media**: Add Images and Videos to the scene, which are automatically displayed on planes.
- **Import Audio**: Import sound files with an audio preview before adding them as spatial sound objects.
- **Import 3D Models**: Load 3D models in GLTF/GLB, OBJ, and STL formats.
- **Export Scene**: Export the entire scene as a GLTF or OBJ file.
- **Project Save/Load**: Save the complete scene state to a `.cyb` file and load it back into the editor.

## UI & Panels
- **Left Toolbar**: Quick access to add shapes, effects, and select primary tools.
- **Object List Panel**: A comprehensive list of all objects in the scene. This panel can be closed and re-opened to maximize viewport space.
- **Tooltips**: Helpful tooltips for all toolbar icons and buttons.

## Special Effects & Advanced Features
- **Particle Systems**: Add pre-configured particle effects like Fire, Rain, Snow, Steam, Magic, Water, and Fog.
- **Skybox**: Enclose the entire scene in a 360-degree sky background.
- **Waypoints**: Place visual map-pin style waypoints within the scene.
- **Time of Day Slider**: Interactively control the scene's lighting by dragging a slider from 0 to 24 hours.
